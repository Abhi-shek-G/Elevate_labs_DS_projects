# Data Science & Machine Learning Project Portfolio

This repository contains a collection of end-to-end data science and machine learning projects focused on solving real-world business problems. Each project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model building, and evaluation.

## 1. No-Show Prediction

### Overview
Predicts whether a customer or patient will miss a scheduled appointment.

### Key Features
- Data cleaning and preprocessing  
- Feature engineering (appointment timing, demographics, waiting period)  
- Classification models: Logistic Regression, Random Forest, XGBoost  
- Evaluation metrics: Accuracy, Precision, Recall, F1-score  

### Business Impact
- Reduces missed appointments  
- Improves scheduling efficiency

## 4. HR Analytics – Predict Employee Attrition

### Overview
Predicts employee attrition to improve retention strategies.

### Key Features
- Data preprocessing and encoding  
- Feature importance analysis  
- Models: Logistic Regression, Decision Tree, Random Forest  
- Evaluation using Confusion Matrix and ROC-AUC  

### Key Factors
- Job satisfaction  
- Salary  
- Work-life balance  
- Tenure  

### Business Impact
- Reduces employee turnover  
- Improves workforce planning

## 5. Customer Lifetime Value Prediction

### Overview
Predicts the total revenue a customer will generate over time.

### Key Features
- Customer segmentation  
- Feature engineering (RFM analysis)  
- Regression and probabilistic models  
- CLV prediction and ranking  

### Techniques
- RFM Analysis  
- Regression Models  
- BG/NBD and Gamma-Gamma models  

### Business Impact
- Identifies high-value customers  
- Optimizes marketing spend  
- Improves retention strategies
